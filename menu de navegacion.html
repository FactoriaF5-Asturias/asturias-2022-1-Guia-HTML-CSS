Menu de Navegacion

Representa una sección de una página cuyo propósito es proporcionar enlaces de navegación, ya sea dentro del documento actual o a otros documentos. Ejemplos comunes de secciones de navegación son menús, tablas de contenido e índices.

¿Como crear un menu de navegacion en HTML?

Primero, crearemos nuestra estructura basica en HTML: dentro de nuestro elemento "body" crearemos un elemento "header", el cual sera una barra de navegacion superior. Dentro colacaremos el elemento "nav" que contendra el menu desplagable en forma de lista, esta lista "ul" tendra como id "menu".
