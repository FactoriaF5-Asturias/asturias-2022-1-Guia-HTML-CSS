Contenedores

HTML proporciona un conjunto de etiquetas de contenedor. Esas etiquetas pueden contener un conjunto no especificado de otras etiquetas

Contenedores html

Dentro de html tenemos diferentes contenedores y puede resultar dificil diferenciar entre cual es más adecuado utilizar. A continuación detallamos diferentes contenedores y su aplicación:

Article:
Es un contenedor de bloques de contenido que se consideran independientes del sitio web y pueden, por lo tanto, ser vistos, reutilizados y distribuidos por separado.
Section:
Es un elemento de seccionamiento genérico, diseñado para contener una parte de un documento temáticamente definida. Los contenidos de un elemento section son usualmente precedidos de un encabezado.
Div:
Sirve para crear secciones o agrupar contenidos.
